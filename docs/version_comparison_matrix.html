关键结论:
1. v1.0→v1.1 将 3 个孤岛转为纯 FP32 (Vbi/Vbseff/Vth k1ox)，零收敛退化——11/11 全部保持。
2. v1.2 的 FAIL 来自精度阈值超标（如 vbs=1.05%），非收敛失败。这些电路在 v1.0/v1.1 中均收敛。
3. 无 FP64 孤岛的纯 FP32 版本预计 8/16 收敛失败：OTA/OpAmp DC 因 Vbseff/k1ox 消去不收敛，TRAN 因 DC 失败无法运行。
4. 6 个 FP64 孤岛是NP-hard 级别的数值必要条件——去掉任何一个都会导致特定电路类型的收敛失效或 NaN。
