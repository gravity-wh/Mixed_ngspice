## 5. FF corner —— 6 器件 DC（收敛失败）

| 参数     | FP64 (.lib, FF) | FP32 (FF model) |
| ------ | --------------- | --------------- |
| V(vb2) | 1.555276 V      | **收敛失败 (NaN)**  |
| V(vb1) | 0.171142 V      | —               |
| V(vb3) | 1.585066 V      | —               |
|        |                 |                 |

**失败原因**：FF corner 下 vth 偏移使偏置点移动，BSIM4 内部计算路径在 FP32 下产生 NaN（与 TT corner 中已修复的 NaN 同源）。已定位根因（全参数模型下 BSIM4 负载函数中的 NaN 传播路径），防火墙方案已验证可行，因代码结构复杂尚未在生产构建中部署。

**影响范围**：TT corner 已完整验证。FF/SS/FNSP/SNFP 需要收敛调优。

## 6. SS / FNSP / SNFP corner —— 收敛失败

同 FF corner 问题——vth/u0 偏移导致的偏置点变化触发了 BSIM4 NaN 路径。属于同一类问题。

## 7. 误差分析总结

### 7.1 误差来源分类

| 误差来源                       | 典型量级        | 影响参数            |
| -------------------------- | ----------- | --------------- |
| SPICE_REAL 截断              | ~1.2e-7 相对  | 所有浮点存储          |
| 数学函数 float→double→float 转换 | < 1e-6 相对   | 仅 exp/log/sqrt  |
| Leff/Weff (double 精度孤岛)    | < 1e-12 相对  | L/W 几何参数        |
| 矩阵求解浮点舍入                   | ~1e-7 × 条件数 | 高阻节点电压          |
| dmcg 移除 (模型简化)             | ~1-10%      | AD/AS/PD/PS 默认值 |
|                            |             |                 |

### 7.2 误差传播路径

```
FP32 输入 (模型参数)
  → BSIM4 温度计算 (Leff/Weff: double精度 → 无误)
    → BSIM4 负载 (Vth, gm, gds: SPICE_REAL算术 → ~1e-7相对)
      → 矩阵装配 (stamp: SPICE_REAL累加 → ~1e-7相对)
        → 稀疏求解 (LU: 条件数~1e3 → ~1e-4相对放大)
          → Newton收敛 (fabs判断 → 精度充足)
            → DC工作点 (< 0.003%误差 @ 关键节点)
```

### 7.3 已验证 vs 未验证的工况

| 工况       | DC (TT)       | DC (FF) | 瞬态 (TT)          | 瞬态 (FF) |
| -------- | ------------- | ------- | ---------------- | ------- |
| 单管 PMOS  | **✅ <0.001%** | —       | —                | —       |
| 6 器件偏置   | **✅ <0.003%** | ❌ NaN   | **✅ 0.18µV RMS** | ❌       |

| 噪声分析     | 🔲            | 🔲      | —                | —       |
| 蒙特卡洛     | 🔲            | 🔲      | —                | —       |

## 8. 精度 vs 精度的权衡

| 维度         | FP64        | FP32             | 代价            |
| ---------- | ----------- | ---------------- | ------------- |
| 机器 epsilon | 2.22e-16    | 1.19e-7          | 9 个数量级        |
| DC 偏置精度    | 基准          | **< 0.003%**     | 可忽略           |
| 瞬态波形精度     | 基准          | **< 0.2 µV RMS** | 可忽略           |
| 收敛鲁棒性      | 全 corner 收敛 | **仅 TT 收敛**      | 3/5 corner 失败 |
| 分析速度       | 基准          | **1.4× faster**  | —             |
| 内存占用       | 基准          | **-45% 工作集**     | —             |

**综合判断**：TT corner 下精度损失完全可以忽略（< 0.003%），收敛鲁棒性是当前最大差距——FF/SS/FNSP/SNFP 需要额外的 NaN 防护投入。
