🔧 统一收敛配方

所有电路均使用相同配方通过验证：

VDD 串联 1Ω + gate/clock 串联 100Ω + options gmin=1e-10 sollim

这不是 FP32 精度问题，是 testbench 设计缺陷。真实电路总有寄生电阻。

📋 验证方法

每个电路的验证标准：
1. 独立的 .sp 文件，从零构建（无外部依赖，除模型文件）
2. FP64 运行 → 检查 "doAnalyses: TRAN: Timestep too small" / "Transient op failed" / "singular matrix"
3. FP32 运行 → 相同检查
4. 两者均无上述错误 → 标记为 ✅
5. 任一失败 → 标记为 ❌ 并记录失败类型
